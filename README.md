# 🐾 Ushki Store Clone - Vue 3 + TypeScript

Клон интернет-магазина аксессуаров для косплея, созданный на **Vue 3** с использованием **TypeScript**.

## 🚀 Технологии

- **Vue 3** (Composition API, `<script setup>`)
- **TypeScript** - строгая типизация
- **Vue Router** - маршрутизация между страницами
- **Pinia** - управление состоянием (корзина товаров)
- **GSAP** - продвинутые анимации появления элементов
- **Vite** - быстрая сборка проекта

## ✨ Особенности

### Анимации
- Плавное появление заголовка и навигации при загрузке
- Анимированное появление карточек товаров с эффектом stagger
- Анимации при наведении на кнопки и карточки
- Плавные переходы между страницами
- Анимация открытия/закрытия меню и корзины
- Плавающая анимация для изображений

### Функционал
- **Главная страница** - Hero секция, преимущества, популярные товары
- **Каталог** - фильтрация по категориям, добавление в корзину
- **О нас** - информация о компании и команде
- **Корзина** - управление количеством товаров, оформление заказа
- **Адаптивный дизайн** - поддержка мобильных устройств

## 📁 Структура проекта

```
src/
├── components/
│   └── layout/
│       └── AppHeader.vue      # Шапка с навигацией и корзиной
├── router/
│   └── index.ts               # Настройка маршрутов
├── stores/
│   └── cart.ts                # Pinia store для корзины
├── views/
│   ├── HomeView.vue           # Главная страница
│   ├── CatalogView.vue        # Каталог товаров
│   ├── AboutView.vue          # Страница "О нас"
│   └── CartView.vue           # Страница корзины
├── App.vue                    # Корневой компонент
├── main.ts                    # Точка входа
└── style.css                  # Глобальные стили
```

## 🛠 Установка и запуск

### 1. Установка зависимостей
```bash
npm install
```

### 2. Запуск в режиме разработки
```bash
npm run dev
```

### 3. Сборка для продакшена
```bash
npm run build
```

### 4. Предварительный просмотр сборки
```bash
npm run preview
```

## 📱 Страницы

1. **Главная (`/`)**
   - Приветственная секция с анимацией
   - Карточки преимуществ
   - Превью популярных товаров

2. **Каталог (`/catalog`)**
   - Фильтрация по категориям (Ушки, Хвостики, Ошейники, Аксессуары)
   - Карточки товаров с кнопкой добавления в корзину
   - Анимация при фильтрации

3. **О нас (`/about`)**
   - Информация о магазине
   - Ценности компании
   - Карточки команды

4. **Корзина (`/cart`)**
   - Список выбранных товаров
   - Управление количеством
   - Подсчёт итоговой суммы
   - Оформление заказа

## 🎯 Новые анимации (в сравнении с оригиналом)

1. **GSAP анимации появления**:
   - Каскадное появление элементов с задержкой
   - Эффекты scale и rotation для изображений
   - Stagger-анимации для списков

2. **Vue Transitions**:
   - Плавное открытие мобильного меню
   - Анимация выезжающей корзины
   - Fade-эффекты для оверлеев

3. **Hover-эффекты**:
   - Подъём карточек при наведении
   - Увеличение кнопок
   - Анимация подчёркивания ссылок

4. **CSS Keyframes**:
   - Плавающая анимация для эмодзи
   - Пульсация элементов
